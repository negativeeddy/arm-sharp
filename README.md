# ARM .NET Port

A .NET port of [Automatic Ripping Machine](https://github.com/automatic-ripping-machine/automatic-ripping-machine) — a daemon that automatically rips and transcodes CDs, DVDs, and Blu-rays.

## Build

```bash
dotnet build
```

## Run

```bash
dotnet run --project src/ArmRipper.WebUi
```

## Test

```bash
dotnet test
```
